Url Shortener
A small project made for Free Code Camp back-end curriculm

What's this?

A simple project where I can pass a url and receive a shorter verion in a JSON response.
When visited, the short URL will automatically redirect me to my original page.

Exeample of usage

/new/my website url
https://urlshort-microservice.herokuapp.com/new/https://www.reddit.com

Output exeample

{"original_url":"https://www.reddit.com","short_url":"urlshort-microservice.herokuapp.com/SJELsDUn"}

then visiting urlshort-microservice.herokuapp.com/SJELsDUn will redirect me to the origian webpage.
